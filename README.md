# looking-for-job

Analysis of vacancies in the labor market

## Technology stack

 - Spring Boot
 - Thymeleaf
 - JasperReports
 - WebJars
 - Tomcat 8

## Spring Features

 - Spring JavaConfig
 - Spring MVC
 - Spring Data
 - Spring JPA
 - Spring JTA

## To run this application

Using __Embedded Apache Tomcat__:

    ./gradlew bootRun

You can then access the sample here: [http://localhost:8080/](http://localhost:8080/)
